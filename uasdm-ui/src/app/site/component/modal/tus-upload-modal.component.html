Radiometric Collection

You are uploading to a radiometric collection. These sensors capture scientific temperature or reflectance data, not standard RGB imagery. Imagery must contain a single band in .tif format — not RGB. Video must be encoded as a singular band — not RGB. Radiometric calibration is automatically enabled (as CAMERA), but if your sensor supports CAMERA+SUN, you may enable that during processing.

Multispectral Collection

You are uploading to a multispectral collection. These sensors capture multiple narrow wavelength bands, not standard RGB imagery. Radiometric calibration is automatically enabled (as CAMERA), but if your sensor supports CAMERA+SUN, you may enable that during processing.

Geo Location File

You are uploading to a sensor which requires a geo location file. This geo location file is generated separately from the rest of the imagery, and is sometimes called 'PIX4D.csv' or 'geo.txt', but may vary by sensor. This file may be uploaded by itself, or inside a zip with the rest of the imagery. If a geo location file already exists, it will be replaced. RX1RII geo location files will be converted to, and stored as, an ODM file-format upon upload. Validation of this file also happens on upload. References to imagery inside this file are case sensitive and must match exactly.
