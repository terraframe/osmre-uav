Public Map Services

IDM hosts both image mosaic and hillshade public OGC compliant map services that are powered by GeoServer.
The public IDM WMS and WMTS map services are accessed by using the appropriate root URL format for the specific service type.

Root WMS URL: https://aim.geoplatform.gov/wms?...
Root WMTS URL: https://aim.geoplatform.gov/gwc/service/wms?...

More detailed information on how to use these services can be found on the GeoServer documentation.
  • GeoServer WMS Documentation
  • GeoServer WMTS Documentation (cached & tiled service)
